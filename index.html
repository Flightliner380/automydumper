Automated MySQL backups.

Automydumper automates taking MySQL backups by providing automatic cron-based backups, retention cleanups and email notifications, amongst other things. It is written in Bash and uses Mydumper 0.9+ as backend.

Versions

The current version of Automydumper is 1.0.
